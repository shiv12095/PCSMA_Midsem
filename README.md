# PCSMA_Midsem
Code base for PCSMA midsem

<B>Setup</B>
<ol>
<li>Modify the database settings for mysql and mongo in application.properties</li>
<li>Clone the repository and open in Eclipse as a maven project</li>
If you are using the terminal, then from the root directory of the project
  <ol>
    <li>To compile the code, run <b>mvn clean install</b></li>
    <li>To run the server, run <b>mvn spring-boot:run</b></li>
  </ol>
</ol>
